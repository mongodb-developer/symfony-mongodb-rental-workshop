## Platform.sh 

Platform.sh is a Platform as a Service (PaaS) that allows you to host, build, and scale applications in the cloud. It is a great tool to deploy and manage [Symfony](https://symfony.com/cloud/) your applications.

## CLI Installation of our project

We will follow the guide on Symfony website to [deploy]9https://docs.platform.sh/guides/symfony/get-started) an exsiting project on Platform.sh.
### Prerequisites
- Git
- Symfony CLI

### Init the project
```bash
symfony project:init
```
